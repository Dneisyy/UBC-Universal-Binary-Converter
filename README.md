# UBC-Universal-Binary-Converter
A "Powerful" Converter, that can convert binary to text and text to binary(with copy result button!!!!)both for linux(any distro) and windows. .
|Ubuntu/Debian/As**a(astra👎)
|sudo apt update && sudo apt install g++
########################################
|Fedora/R*d OS (red os👎)
|sudo apt update && sudo apt install g++
########################################
|Arch/Manjaro
|sudo pacman -S gcc
########################################
|Convert Linux
|g++ UBC-linux.cpp -o UBC-linux `pkg-config --cflags --libs gtk+-3.0` -static-libgcc -static-libstdc++
|'-static' is optio(a)nal
